# Restart Ukraine Women Platform

Цифровий MVP платформи професійного повернення та економічної адаптації українок.

## Живий сайт

https://barik2070.github.io/Restart-Ukraine-Women-Platform/

## Реалізовано

- адаптивна головна сторінка;
- сторінки програм і трьох навчальних треків;
- практичний AI-модуль;
- трикрокова анкета учасниці;
- сторінка для партнерів і донорів;
- прототип особистого кабінету;
- UX-концепція Telegram-бота;
- UI-kit, moodboard і design guide.
